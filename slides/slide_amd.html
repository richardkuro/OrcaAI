Self-hosted LLM
vLLM + Llama 3 70B on ROCm
Replace Gemini API calls in ai.js with a local vLLM server running Llama 3 70B (or Mistral). The endpoint shape mirrors the current callGemini() function — only the URL changes. Full summarization, key points, and tone analysis with no API costs.
